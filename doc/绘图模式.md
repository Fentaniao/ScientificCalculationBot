# 绘图模式

绘图功能的每个参数前后务必加上引号

### 一元函数

draw(fun, x_arange):

```
# 画一元显函数
draw_fun('sin(x) / x', '-10，10，0.1')
```

### 隐函数

draw_imp(fun, x_arange,y_arange):

```
# 画二元隐函数
draw_imp('17 * x**2 -16*abs(x)*y + 17 * y**2-256','-6,6','-6,6')
```

### 参数方程

draw_para(x_eq, y_eq, t_arrange)

```
# 画单参数的参数方程
draw_para('2*sin(t)','3*cos(t)','0, 2*pi, 0.1')
```
